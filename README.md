# .github repository of Nerd-Monke-LLC GitHub organization

1. This is an admin repository used for distributing common issue templates, labels, and workflows across the Nerd-Monke-LLC GitHub organization.

2. The README in the `/profile` directory will appear on the organization's profile. [Edit Organization profile README](https://github.com/Nerd-Monke-LLC/.github/edit/main/profile/README.md).

Issues in this project are disabled.
If you want to report an issue with an Nerd-Monke-LLC projects or products and you ended up here, please try to find the corresponding GitHub project where you can report your issue, such as:

- "Filler"
- "Filler"

... or [many more](https://github.com/orgs/Nerd-Monke-LLC/repositories) and report your issue there.
